# Brain Games

«Игры разума» — набор из пяти консольных игр, построенных по принципу популярных мобильных приложений для прокачки мозга. Каждая игра задает вопросы, на которые нужно дать правильный ответ. После трех правильных ответов игра считается пройденной. При неправильном ответе игра завершается.

## Игры
1. **Проверка на четность:** `brain-even`
2. **Калькулятор:** `brain-calc`
3. **НОД:** `brain-gcd`
4. **Арифметическая прогрессия:** `brain-progression`
5. **Простое ли число:** `brain-prime`

## Установка

Для установки пакета глобально в систему:
```bash
npm install -g @hexlet/code
```

Если вы устанавливаете локально из репозитория:
```bash
make install
npm link
```

## Использование

Для запуска любой из игр введите соответствующую команду в терминале:

```bash
brain-games      # Приветствие
brain-even       # Игра "Проверка на чётность"
brain-calc       # Игра "Калькулятор"
brain-gcd        # Игра "НОД"
brain-progression# Игра "Арифметическая прогрессия"
brain-prime      # Игра "Простое ли число"
```
